# Análisis del Dominio: Consultoría Empresarial "Soluciones Estratégicas"

## 1. Decisiones de Diseño
- Los atributos para la clase `ConsultingProject` se eligieron con el objetivo de establecer la identidad y datos fundamentales para cada proyecto: `projectId`, `projectTitle`, y `projectBudget ` fueron seleccionados por ser los datos más básicos y representativos del proyecto de consultoría.

- **Aclaración sobre projectBudget:** Este atributo se escogió porque representa el valor económico total del contrato que el cliente pagará por el servicio.
- **¿Por qué elegí estos atributos?**
  En esta actividad escogí estos tres atributos, ya que se me hicieron generales y que debería tener cada proyecto de consultoría, como lo son su ID, nombre del área y el presupuesto o dinero.
- **¿Qué validaciones implementé?**
  Implemente soporte con IA, videos acerca de empresas relacionadas, así como documentación en páginas web para tener un contexto y bases claras.

## 2. Dificultades Encontradas
- La primordial fue la elección de los atributos, debido a que no tenía el conocimiento de que iba por buen camino o que sí correspondían a una consultoría o a cada proyecto de este tipo de empresa.
